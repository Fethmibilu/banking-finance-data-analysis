# Banking & Finance Data Analysis Project

## Overview
This project focuses on cleaning, validating, and analyzing banking data using Excel and Power BI.

## Tools Used
- Microsoft Excel
- Power BI

## Project Tasks
- Data Cleaning
- Missing Value Handling
- Duplicate Removal
- Data Validation
- Dashboard Development

## Dashboard KPIs
- Total Customers
- Customer Segment Analysis
- KYC Status Distribution
- Credit Score Analysis
- Monthly Income Analysis

## Key Insights
- Retail customers form the largest customer segment.
- Most customers are KYC verified.
- Customer income and credit scores show strong patterns across segments.
